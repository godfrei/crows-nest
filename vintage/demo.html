Dark Forces Demo

What's that? You haven't played Dark Forces yet? My first response would be to ask what rock you've been living under for the last couple years, but I'll forgo that in exchange for my second: download the demo from LucasArts to get a taste of what you're missing. For your convenience I've linked to the files, both Mac and PC formats. Just take a quick scan through the information below, pick the files you need, and you're on your way.

Here's what you need to know to get started:

System Requirements

NOTE: These are for the demo and game only. Several of the add-on levels require more memory and recommend faster processors.

PC
CPU 386/33 DX or higher (486 strongly recommended; I've found you need to decrease the screen size otherwise)
Memory(RAM) 8M system memory (6.5M free)
Operating System MS-DOS 5.0 or higher
Input Device A mouse is required, you can use a joystick too
Video VGA(local bus or PCI recommended)
Sound Cards Sound Blaster, Sound Blaster 2.0, Sound Blaster Pro, Sound Blaster 16, AWE 32, AdLib, Roland MT 32/LAPC-1, Pro Audio Spectrum. General MIDI and compatibles also supported

Macintosh
CPU 68040 or better
Memory(RAM) 4.5M free required, 7M preferred
Monitor 256 color monitor

If you meet these requirements then you're ready to go. For the PC there are three files you can get(for the Mac there is just one). Only the first is required to actually play the level; the other two are the opening and trailing cutscenes. I'd recommend them if you can spare the space and have extra time for downloading them.

DFORCES1.ZIP: This file contains the first level of Dark Forces. This is the only file that is required to play the demo.
File Size: 3.3 Megabytes
HD Space: 8.0 Megabytes

DFORCES2.ZIP: This file contains the opening cutscene for the demo. This file is optional. The demo will work without this file, however you will not see the opening cutscene.
File Size: 1.0 Megabytes
HD Space: 2.0 Megabytes

DFORCES3.ZIP: This file contains the trailing cutscene for the demo. This file is optional. The demo will work without this file, however you will not see the trailing cutscene.
File Size: 5.8 Megabytes
HD Space: 8.0 Megabytes
